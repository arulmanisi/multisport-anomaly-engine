# Roadmap
Living roadmap; update as priorities change.

## Near term (0-4 weeks)
- Define data inputs and availability.
- Stand up ingestion pipeline and storage.
- Choose baseline anomaly techniques and evaluation metrics.

## Mid term (1-3 months)
- Iterate on anomaly detection models with labeled signals.
- Build minimal API/CLI to surface anomalies.
- Add monitoring and alerting hooks.

## Long term
- User feedback loop for labeling and triage.
- Integrations with dashboards or ops tools.
- Performance tuning and cost optimization.
